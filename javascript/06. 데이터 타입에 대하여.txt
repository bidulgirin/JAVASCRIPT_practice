>데이터 타입에 대하여

데이터 타입이란?

	 값의 종류를 뜻한다

종류는 숫자, 문자열, 불리언, undefined, null, 심벌 타입 등이 있다

>숫자 타입 

숫자 타입은 배정밀도 64비트 부동소수점 형식을 따른다 
즉, 모든 수를 실수로 처리하고 정수만 표현하기위한 데이터 타입은 없어!
2진수로 메모리에 저장이 된다!!!


Infinity -Infinity NaN으로도 표현 할수 있다

NaN을 다르게 표기하면 값이 아닌 식별자로 인식함

>문자열 타입 

텍스트를 데이터를 나타내는 데 사용한다.
'' "" `` 으로 텍스트를 감싼다 보통 ''(작은 따옴표) 로 감쌈

문자열은 원시 타입이며 변경 불가능한 값이다 이것은 문자열이 생성되면 
바꿀 수 없다는 것을 의미 한데!

여기서 백틱은 뭐지? > 템플릿 리터럴 (ES6버전이상인 브라우저만)


템플릿 리터럴은 멀티라인 문자열과 표현식 삽입 태그드 탬플릿등 문자열 처리 기능이다

멀티라인 문자열: 일반적으로 문자열을 적을때 줄바꿈이 허용되지 않는데
이걸 가능하게 하고 공백도 그대로 표기되는 것이  ``(백틱) 
일반 문자열에서 줄바꿈이 가능하도록한 문자들도 존재하지만 백틱을 사용하면
가독성이 용이하여 좋다~~

표현식 삽입: 일반적으로 연산자는 + 연산자를 이용해 동작하는데 

${ } 으로 표현식을 감싸면 그 안에 들어간 모든 표현식은 문자열로 표기되어 강제 삽입된다!

>불리언 타입 

참 거짓을 나타내는 타입이다
 흐름을 제어하는 조건문에서 자주 사용한다

>undefined 타입 

변수는 암묵적으로 undefined로 초기화 된다! 값이 할당되지 않은 변수에게
undefined가 메모리로 부터 반환 되어 온다

의도적으로 defined를 할당하는것은 바람직하지 않다 
그 대신 null을 할당하자!

>null타입 

변수에 값이 없어요를 의도적으로 나타낸것임~

만약 사용하면 명시적으로 제거하는 것을 의미 하고 가비지 콜렉션을 수행한다고 한다
유효한 값을 반환할 수 없는 경우도 null을 반환한다

>심벌 타입(아직도 이해잘 안감ㅋㅋㅋ) 
변경 불가능한 원시 타입의 값이다 .
Symbol 함수를 사용해 호출 생성한다. 다른 값과 중복되지 않는 유일한 값!!!

>객체 타입 

자바 스크립트를 이루고 있는 모든 것이 객체 이고 그 나머지 6개 정도는 원시타입이다

> 데이터 타입 대체 뭐길래 딱딱 나누는 걸까???

	1. 값을 저장할때 확보해야 하는 메모리 공간의 크기를 결정하기위해
	2. 값을 참조할 때 한 번에 읽어 들여할 메모리 공간의 크기를 결정하기 위해
	3. 메모리에서 읽어 들인 2진수를 어떻게 해석할지 결정하기 위해

>동적 타입과 정적 타입? 

정적타입: c언어와 같은 언어인데 변수를 선언하고 변수의 타입을 사전에 선언해야하는 
명시적 타입 선언 이라 한다. 타입 체크를 하여 런타임에 발생하는 에러를 줄인다!! 

동적타입: 자바스크립트 언어와 같은 것인데
 변수를 선언하고 값을 할당하면 바로 변수가 정해져 버린다 
언제든지 타입이 변경 될 수 있어서 동적타입 언어라고 하는 것이다.
