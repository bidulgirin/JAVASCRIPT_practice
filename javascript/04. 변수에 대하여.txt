<모던 자바 스크립트 Deep Dive 4장 변수>

>변수가 뭐고 왜 사용하나요?

-실행문을 작성했는데 결과 값을 가지고 싶은데 갖을 수 없어서 메모리에 접근해야하는데
직접 메모리에 접근하면 위험해서 저장소를 하나 만들어서(가방) 필요할때마다 꺼내 쓸수
있도록 이름을 적어두는 식별자 중 하나를 변수라고 한다~

-식별자는 뭐 뭐가 있냐면 변수, 함수, 클래스 등이 있단다~

>함수 선언 방법에 대하여

함수 선언을 하려면 var.const, let 이라는 키워드가 쓰이는데 

const 랑 let은 var의 단점때문에 나온 애들이다 (ES6부터 지원됨)

함수 선언은 var 변수명; 이런 식으로 선언 하고 

var 변수명 = 값;   =이라는 연산자를 이용하여 값을 할당한다 

그리고 실행은 변수 선언 먼저 확인 하고 변수 값을 확인한다~

> const는 뭐야? var이랑 차이점?

const는 재할당이 불가능한 변수 선언이래 원래 var은 재할당이 가능해서 가비지 콜랙터가
사용하지 않은 변수를 수거해갔는데 그런 과정이 아예 사라지겠는 걸?

> 언 매니지드 언어와 매니지드 언어


언 매니지드 언어 :c 언어와 같음
-개발자가 메모리를 할당하고 해제할수 있음 / 치명적 오류 발생가능성 높음

매니지드 언어: 자바스크립트와 같음
-개발자는 메모리와 관련된 모든 기능을 제어할수 없음 안전하지만 성능면에서는 손실 

>식별자 규칙에 대해서 

1. 문자 언더스코어(_) 달러기호($)로 시작해야한다/숫자는 안됨!! 
2. 예약어 안됨
3. 가독성을 높이기 위해 의미를 명확히 표현해야한다.
4. 대소문자가 구분된다!

대부분 카멜케이스 (낙타 표기법이라고도 한다 두번째 문자부터 대문자 사용) 를 사용하고 
생성자 함수, 클래스 함수에는 파스칼 케이스를 사용한다(단어 첫글자 마다 대문자로 표기)
그외에 스네이크케이스 도 있다 (final_case_)

가독성을 위해 적는거래~
